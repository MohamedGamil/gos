Gems/SASS | Documentation

Gems-Over-SASS is a small framework that helps you when you start coding up your Web Designs using a set of HAML, SASS, and Javascript files and libraries to get the job done faster and easier.

Features

  • Solid yet Easy-To-Deal-With files structure.
  • Build all -or most- of your final CSS into 1 file.
  • Wide array of SASS Mixins and Functions that will save you hours of work.
  • New SASS functions that will help you to easily manipulate strings -Read the Documentation-.
  • Deal with Web Fonts even more easily.
  • Easily structure your document using either Bootstrap Fixed, or Responsive Grid systems -Supports Fluid Grid Too-.
  • Easily include your Responsive styles.
  • Easily create your Typography styles.
  • Easily create Right-To-Left styles.
  • Easily implement Multi-Colored Themes.
  • Apply Bootstrap Icons styles -Glyphicons- directly to any element.
  • Comes with a Powerful -Reset.css-.
  • Cross-Browser Compatibility.

Getting Started

  • The best way to use GOS, is to use PrePros to compile your source code.
  • After installing PrePros, Drag and Drop your cloned repository to PrePros Application.
  • Make sure to uncheck "Full Compass Support" for "css/main.scss".
  • Open "index.haml", "css/main.scss", and all files in "css/sass-user/".
  • Type your name, and copyrights at the top of "css/main.scss", and you're ready to go..
